# SocialMediaChromeExtension

This is a chrome extension that closes a social media page on your chrome app if you spend more than 5000 seconds on a social media page.
The social media pages are instagram, facebook or twitter. The main goal of the extension is to save on ones time by preventing one 
from spending so much time on social media pages.

Before it closes the page it first alerts you.

The extension is available on developer mode and can be used when you switch to developer mode on your chrome.

To see the extension in action
1) clone the repo

```
https://github.com/margaretmwaura/SocialMediaChromeExtension.git

```

2) Go to chrome://extensions/
3) Click on the button load unpacked and select the location where you cloned the repo in.
