# SocialMediaChromeExtension

This is a chrome extension that closes a social media page on your chrome app if you spend more than 5000 seconds on the social media page.
The social media pages are instagram, facebook and twitter. The main goal of the extension is to save on ones time by preventing one 
from spending so much time on social media pages.

Before it closes the page it first alerts you.

The extension is available on developer mode and can be used when you switch to developer mode on your chrome.
